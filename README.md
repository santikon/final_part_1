8. Codeigniter ใช้ง่ายไม่ต้องใช้คำสั่งใน cmd แต่ rails ต้องติดตั้งหลายอย่างและมีขั้นตอนหลายอย่างที่สำคัญใช้คำสั่งใน cmd
